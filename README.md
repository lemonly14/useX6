## 环境和依赖

- node
- npm
- vite
- eslint
- vue 3.x 全家桶

## 项目下载和运行

- 拉取项目代码

```bash
git clone https://github.com/lemonly14/useX6.git
cd useX6
```

- 简介

```
src/views/register.ts 注册自定义cell

src/views/useX6.ts 封装方法

```

- 安装依赖

```
npm install
```


- 开发模式运行

```
npm run dev
```

- 编译项目

```
npm run build
```
